# Description

Desjardins Backend technical test

## Getting Started

1. Install
```sh
mvn clean install
```

2. Serve the Application
```sh
mvn spring-boot:run
```

The backend is now running and accessible at http://localhost:8080.

## Technical Stack

* Java 17
* Spring boot
* H2 Database
* Lombok
* Maven

## Improvements
* Do not save if the same actor is saved twice
